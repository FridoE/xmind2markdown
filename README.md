# xmind2markdown

xmind2markdown converts XMind mindmap files to markdown format. I created it to migrate from xmind to Obisidan. 

## Features
- converts mindmap leaves to markdown headlines (`#`, `##`, …)
- (limited) support for XMind icons/markers (e.g. task status, priority, ...)
- supports the XMmind 2020+ format (JSON)

## Installation
```bash
pip install xmind2markdown

## Usage
```bash
```
```
xmind2md input.xmind output.md
